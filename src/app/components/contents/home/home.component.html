WHAT IS SCOUTER ?


A generic system that helps in capturing, analyzing, scoring and storing the contextual information of a given application domain. By leveraging on the power of semantics and machine learning, Scouter aims at providing a powerful set of tools to enrich the pipeline of any given Big Data Platform.


PLUG-IN DESIGN

Scouter is based on Java, is extensible, and provides a plug-and-play environment that makes it a flexible base for rapid prototyping and application development.
MACHINE LEARNING

Scouter offers a set of ML functions that can adapt to any Big Data platform whatever the engine used and without losing the capability to process various types of data sources.
NLP FUNCTIONS

Leveraging on the flexibility of ontologies and the possibility to extend it based on other research work, Scouter offers a powerful list of nlp functions to support business decisions.
